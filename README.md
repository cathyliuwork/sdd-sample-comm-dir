## 什么是微信群成员介绍？

**核心价值**：解决微信群成员介绍信息散乱、难以管理和长期保存的问题。

本产品是一个**移动优先的 Web 应用**，帮助社区管理者：
- 结构化收集群成员介绍信息
- 生成美观的可复制文字内容，方便分享到微信群
- 为每个社区提供独立的成员列表页面
- 支持管理员管理多个社区和成员数据

**解决的痛点**：
- 群成员的介绍（free text message）缺乏规范
- 后进群的人看不到前面的介绍信息
- 介绍多了无法管理，不利于长期分享

**解决方案**：
- Web 应用收集结构化信息（名字、所在地、职业等）
- 生成可复制的格式化文字，便于粘贴到微信群
- 提供公开访问的成员列表页面（按社区隔离）
- 管理员后台统一管理所有社区

## 核心功能

- **社区管理**（管理员） - 创建和管理多个社区，为每个社区生成独立链接
- **成员信息收集** - 结构化表单（姓名、所在地、职业/行业、正在做的事情等）
- **分享内容生成** - 一键生成格式化文字，可复制到微信群
- **成员列表查看** - 公开访问的成员展示页面（按社区）
- **管理员后台** - 查看所有社区总表，按社区筛选，管理成员数据

## 技术栈

- **前端**: Next.js 14 + React 18 + Tailwind CSS
- **后端**: Next.js API Routes + Prisma ORM
- **数据库**: Supabase (PostgreSQL)
- **认证**: 管理员配置文件静态登录，普通用户无需认证
